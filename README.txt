To Install:

1. (If downloaded as zip) Extract the contents of this zip folder into their own folder

2. Open the Extensions page (chrome://extensions/ or about:debugging in firefox)

3. (In Chrome) Tick the Developer Mode

4. Click Load Unpacked Extension (Load Temporary Add-on in firefox)

5. Select the extracted folder (Open the folder and select manifest.json in firefox)

6. (In Chrome) Turn off Developer Mode again.



Optional Steps:

Tick the Allow In Incognito box to use it in incognito mode.

(In Chrome) At the bottom of the page, click Keyboard Shortcuts, and add a shortcut to open the popup.


Note: In Firefox, you will have to reinstall the plugin every time you start a new browsing session.
