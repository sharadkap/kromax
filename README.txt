To Install:

1. Extract the contents of this zip folder into their own folder

2. Open the Extensions page (chrome://extensions/)

3. Tick the Developer Mode

4. Click Load Unpacked Extension

5. Select the extracted folder

6. Turn off Developer Mode again.



Optional Steps:

Tick the Allow In Incognito box to use it in incognito mode.

At the bottom of the page, click Keyboard Shortcuts, and add a shortcut to open the popup.
